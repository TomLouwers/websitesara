# Pedagogical Review: Domain Reference Prompt Templates

## Expert Review Date: 2025-12-30
**Reviewer Profile:** Senior Education Expert - Dutch Primary Education, SLO Framework, CITO Assessment

---

## Executive Summary

This document provides a comprehensive pedagogical review of the AI-ready prompt templates across all 13 domain reference files. The review identifies improvements to ensure alignment with:
- SLO Kerndoelen 2006 framework
- CITO-style assessment methodology
- Dutch primary education didactics
- Age-appropriate pedagogy (Groep 1-8)

---

## Key Improvements Implemented

### 1. **Kerndoel Reference Accuracy**

**Issue:** Inconsistent and sometimes incorrect kerndoel references
- Usage of non-standard terms like "Nieuwe N1", "Nieuwe R4"
- Missing specific kerndoel numbers from SLO 2006 framework

**Solution:**
- All kerndoelen now reference official SLO 2006 numbering (K1-K58)
- Mapping table created for domains:
  - **Rekenen:** K23 (Automatiseren), K28 (Getalsysteem), K29 (Hoofdrekenen), K30 (Cijferend rekenen)
  - **Nederlands Lezen:** K1 (Mondeling), K2 (Schrijven), K3 (Lezen - literair), K4 (Lezen - informatief), K5 (Leren leren), K6 (Studerend lezen), K7 (Kritisch lezen)
  - **Oriëntatie:** O1-O15 (Kerndoelen 37-51)

### 2. **CITO-Style Assessment Alignment**

**Issue:** Lack of specific CITO methodology references

**Improvements:**
- Added "CITO-methode" specifications where appropriate
- Standardized multiple choice to **exactly 4 answer options** (CITO standard)
- Specified independent work requirements ("zelfstandig werken")
- Added time constraints for tempo-based exercises (DMT, AVI)
- Included difficulty level indicators (1F, 1S, 2F for Groep 8)
- Referenced CITO question types:
  - **Letterlijk begrip** (literal comprehension)
  - **Interpretatief begrip** (interpretive comprehension)
  - **Kritisch begrip** (critical comprehension)

### 3. **Age-Appropriate Pedagogy**

#### Groep 1-2 (Ages 4-6)
**Improvements:**
- Emphasis on **concrete, visual, playful** approaches
- Added "met materiaal" (with manipulatives) specifications
- Included "spelend leren" (learning through play) contexts
- Audio/visual support requirements
- Teacher-guided vs. independent work distinctions

#### Groep 3-4 (Ages 6-8)
**Improvements:**
- Transition from concrete to semi-concrete
- Added scaffolding instructions ("hulp waar nodig")
- Progressive independence requirements
- Clear M→E progression with skill building

#### Groep 5-6 (Ages 8-10)
**Improvements:**
- Strategic thinking emphasis
- Metacognitive prompts ("hoe heb je dit opgelost?")
- Multiple solution strategies
- Increased complexity and text length

#### Groep 7-8 (Ages 10-12)
**Improvements:**
- Independent problem-solving focus
- Complex multi-step problems
- Critical thinking and analysis
- Preparation for secondary education (CITO eindtoets)

### 4. **Dutch Educational Context**

**Cultural & Contextual Improvements:**
- Dutch money context (euro, centen)
- Dutch measurements (meter, liter, kilogram)
- Dutch geography (provincies, steden, landschappen)
- Dutch history (10 tijdvakken)
- Seasonal references (Sinterklaas, Koningsdag)
- School contexts familiar to Dutch children

**Didactic Materials Referenced:**
- **Rekenrek** (arithmetic rack)
- **MAB-materiaal** (multi-base arithmetic blocks)
- **Kralenrek** (bead frame)
- **Getallenlijn** (number line)
- **Splitsbord** (decomposition board)
- **Telraam** (abacus)

### 5. **Terminology Standardization**

**Standardized Dutch Didactic Terms:**

**Rekenen:**
- **Startsom** → basis calculation to start with
- **Keersommen** → multiplication tables
- **Deelsommen** → division problems
- **Families** → number families (e.g., 3+4=7, 4+3=7, 7-3=4, 7-4=3)
- **Splitsend rekenen** → decomposition strategy
- **Bruggetje van 10** → bridging through 10
- **Handig rekenen** → efficient calculation strategies
- **Kolomsgewijs** → column-based (for written algorithms)

**Lezen:**
- **AVI-niveau** → Reading level system (Start, M3, E3, M4, E4, M5, E5, Plus)
- **DMT** → Drie Minuten Test (Three Minute Test) for reading fluency
- **Woordenschat** → vocabulary
- **Begrijpend lezen** → reading comprehension
- **Technisch lezen** → technical reading (decoding)

**Assessment Terms:**
- **Referentieniveaus:** 1F (fundamenteel), 1S (streefniveau), 2F (gevorderd)
- **Tussendoelen** → intermediate goals
- **Eindniveaus** → end-of-year levels

### 6. **Assessment Specifications**

**Added to Prompts:**
- **Scoring criteria:** "met correctiemodel" (with answer key)
- **Time limits:** "binnen 3 minuten" for tempo exercises
- **Independence level:** "zelfstandig werken" vs. "met hulp van de leerkracht"
- **Difficulty indicators:** "basisniveau", "verdieping", "extra uitdaging"
- **Format specifications:** exact number of answer choices, layout requirements

### 7. **Progressive Complexity (M → E Level)**

**Strengthened Progression:**

**Midden (M) Level:**
- Introduction of new concepts
- Scaffolded support
- Smaller number ranges
- Visual support emphasized
- "Met hulp" (with assistance) acceptable

**Eind (E) Level:**
- Mastery and application
- Independent work
- Larger number ranges
- Less visual support
- "Zelfstandig" (independently) required
- Integration with other skills

**Example Progression:**
- **3G1 (M):** Tellen tot 100 met sprongen van 2, 5, 10 (counting to 100 with steps)
- **3G4 (E):** Getallen tot 100 lezen, schrijven, vergelijken (reading, writing, comparing)

### 8. **Engagement & Motivation**

**Added Elements:**
- **Concrete contexts:** speeltuin, dierentuin, winkel, sport
- **Story problems:** met verhaalcontext die kinderen aanspreekt
- **Game elements:** spelletjes, uitdagingen, puzzels
- **Variation:** afwisselende vraagtypen om motivatie hoog te houden
- **Success orientation:** passende moeilijkheidsgraad voor succeservaringen

### 9. **Special Populations Considerations**

**Differentiation Indicators:**
- **Extra ondersteuning** → for students needing more help
- **Verdieping** → for advanced students
- **Visuele ondersteuning** → for visual learners / EAL students
- **Concreet materiaal** → for hands-on learners
- **Stappenplan** → step-by-step plans for executive function support

### 10. **Quality Control Standards**

**All Prompts Must Include:**
1. ✅ Exact grade level (Groep X)
2. ✅ Timing (midden/eind schooljaar)
3. ✅ Accurate kerndoel reference(s)
4. ✅ Specific inhoudslijn
5. ✅ Clear learning objective
6. ✅ M/E level specification
7. ✅ Concrete question type examples (minimum 3)
8. ✅ Format specification
9. ✅ Number of exercises to generate
10. ✅ Dutch educational context

**Enhanced Elements:**
11. ✅ Scaffolding/support level
12. ✅ Materials needed (if applicable)
13. ✅ CITO-style formatting (where relevant)
14. ✅ Differentiation options
15. ✅ Assessment criteria hints

## Domain-Specific Improvements

### Rekenen/Wiskunde

**Key Changes:**
- Added references to "rekenrek", "MAB-blokken", "getallenlijn"
- Specified calculation strategies (splitsen, bruggetje, compenseren)
- Added "tafelkaarten" for multiplication practice
- Included "cijferalgoritme" specifications for written work
- Referenced Dutch referentieniveaus for Groep 8

**Example Improvement:**

**Before:**
```
"Genereer 10 oefeningen voor Groep 4. Tafels 1-10. Vraagtypen: 7×8=?. Format: sommen."
```

**After:**
```
"Genereer 10 oefeningen voor Groep 4 (midden schooljaar). Kerndoel: K23 (Automatiseren van rekenfeiten).
Inhoudslijn: Getallen - Bewerkingen - Vermenigvuldigen. Leerdoel (4G1): Alle tafels tot 10 automatisch
beheersen binnen 3 seconden per som. Niveau: M (midden groep 4). Gebruik tafelkaarten methode.
Vraagtypen: 7×8=?, 6×9=?, 8×4=?, ook deelsommen 56:7=?. Mix alle tafels 1-10 door elkaar.
Format: snelle automatiseringsoefeningen, 4 antwoordopties per vraag, CITO-stijl. Zelfstandig werken."
```

### Nederlands

**Key Changes:**
- Precise AVI-level specifications with tempo requirements
- DMT references with words-per-minute targets
- Text type specifications (narratief, informatief, instructief, argumentatief)
- Leesstrategieën explicitly named (SLOP, STAF, voorspellen, visualiseren)
- Length requirements for texts (aantal woorden/zinnen)

**Example Improvement:**

**Before:**
```
"Genereer 10 oefeningen voor Groep 5. Begrijpend lezen. Vraagtypen: wie, wat, waar."
```

**After:**
```
"Genereer 10 oefeningen voor Groep 5 (eind schooljaar). Kerndoel: K4 (Informatief lezen).
Inhoudslijn: Lezen - Begrijpend lezen - Interpretatief niveau. Leerdoel (5L5): Tussen de regels
door lezen, conclusies trekken uit impliciete informatie. Niveau: E (eind groep 5). Gebruik teksten
van 150-200 woorden op AVI E5 niveau. Vraagtypen: Wat kun je hieruit afleiden?, Hoe voelt de persoon
zich (niet letterlijk benoemd)?, Waarom doet iemand dit?, Wat betekent deze zin in de context?.
Format: informatieve of narratieve teksten met 5 meerkeuzevragen (4 opties), mix van letterlijk
(30%) en interpretatief (70%). Volgens CITO-methode begrijpend lezen."
```

### Oriëntatie

**Key Changes:**
- Age-appropriate civic concepts
- Dutch political/geographical context
- Concrete, observable examples for younger children
- Progressive abstraction for older students
- Links to current events (appropriate for age)

**Example Improvement:**

**Before:**
```
"Genereer 10 oefeningen voor Groep 8. Democratie. Vraagtypen: wat is democratie."
```

**After:**
```
"Genereer 10 oefeningen voor Groep 8 (midden schooljaar). Kerndoel: O2-O3 (Kerndoelen 38-39:
Politiek-democratie en Sociale ontwikkeling). Inhoudslijn: Oriëntatie - Mens en Maatschappij -
Verkiezingen en democratisch proces. Leerdoel (8MM2): Begrijpen hoe verkiezingen werken in Nederland.
Niveau: M (midden groep 8). Gebruik concrete voorbeelden van Tweede Kamerverkiezingen.
Vraagtypen: Vanaf welke leeftijd mag je stemmen?, Hoe wordt een stembiljet ingevuld?,
Hoe worden zetels verdeeld?, Waarom is een verkiezingscampagne belangrijk?, Wat is het verschil
tussen coalitie en oppositie?. Format: 10 meerkeuzevragen (4 opties) met actuele Nederlandse context.
Optioneel: gebruik een fictieve schoolverkiezing als herkenbare situatie."
```

## Specific Template Patterns

### Pattern 1: Early Childhood (Groep 1-2)
```
Genereer [aantal] oefeningen voor Groep [1-2] (eind schooljaar).
Kerndoel: [K-nummer].
Inhoudslijn: [domein - specifiek].
Leerdoel ([code]): [beschrijving].
Niveau: E (eind groep [1-2]).
Gebruik concreet materiaal en visuele ondersteuning (plaatjes, foto's).
Speelse context: [thema].
Vraagtypen: [3-4 concrete voorbeelden].
Format: Visueel met 2-4 antwoordopties, grote duidelijke afbeeldingen.
Ondersteuning: Met begeleiding van leerkracht, voorlezen van instructies.
```

### Pattern 2: Building Skills (Groep 3-6)
```
Genereer [aantal] oefeningen voor Groep [3-6] ([midden/eind] schooljaar).
Kerndoel: [K-nummer] ([naam kerndoel]).
Inhoudslijn: [domein - specifiek onderwerp].
Leerdoel ([code]): [precieze beschrijving].
Niveau: [M/E] ([midden/eind] groep [X]).
Gebruik [materiaal/hulpmiddel indien nodig].
Context: [Nederlandse herkenbare situatie].
Vraagtypen: [4-5 gevarieerde voorbeelden met moeilijkheidsopbouw].
Format: [specifiek format met aantal opties].
Werkvorm: [zelfstandig/met ondersteuning].
Differentiatie: [optie voor extra uitdaging of ondersteuning].
```

### Pattern 3: Mastery & Application (Groep 7-8)
```
Genereer [aantal] oefeningen voor Groep [7-8] ([midden/eind] schooljaar).
Kerndoel: [K-nummers, meerdere indien relevant].
[Indien groep 8 E: Referentieniveau: 1F/1S/2F].
Inhoudslijn: [domein - specifiek onderwerp - niveau].
Leerdoel ([code]): [complexe vaardigheid beschrijving].
Niveau: [M/E] ([midden/eind] groep [7-8]).
Complexiteit: [indicatie van denkniveau/stappen].
Context: [realistische, uitdagende Nederlandse situatie].
Vraagtypen: [5-6 diverse voorbeelden, mix van reproduceren, toepassen, analyseren].
Format: CITO-stijl met 4 antwoordopties, mogelijk met meerdere stappen.
Werkvorm: Zelfstandig werken, toetsconditie.
[Voor groep 8 E: Voorbereiding op CITO eindtoets, representatief voor [niveau]].
```
