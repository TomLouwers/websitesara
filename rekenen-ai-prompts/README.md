# Rekenen AI Prompts

Een uitgebreide bibliotheek van AI prompts voor het genereren van rekenopdrachten voor groep 3 t/m 8.

## Overzicht

Dit project biedt een gestructureerde collectie van prompts voor het genereren van rekenopdrachten, georganiseerd per groep en rekendomein.

## Installatie

```bash
pip install -r requirements.txt
```

## Gebruik

Zie de [Gebruikshandleiding](docs/GEBRUIKSHANDLEIDING.md) voor gedetailleerde instructies.

## Structuur

- `prompts/` - Prompt bestanden georganiseerd per groep en onderwerp
- `src/` - Python broncode voor prompt library en generators
- `tests/` - Unit tests en test fixtures
- `examples/` - Voorbeeldscripts voor veelvoorkomende use cases
- `scripts/` - Utility scripts voor beheer en validatie
- `docs/` - Uitgebreide documentatie
