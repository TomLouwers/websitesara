- **Groep 4:** Focus on literal comprehension, simple inference
- **Groep 5:** Add text structure awareness, compare perspectives
- **Groep 6:** Critical thinking, source reliability
- **Groep 7:** Argumentation analysis, bias detection

**Error types:**
- `letterlijk_gemist` - Missed literal information
- `inferentie_fout` - Failed to infer
- `vocabulaire` - Vocabulary confusion
- `structuur` - Text structure misunderstanding
- `detail_vergeten` - Forgot important detail

### Math Exercises

**Common patterns by grade:**
- **Groep 4:** Basic operations, concrete examples
- **Groep 5:** Larger numbers, fractions introduction
- **Groep 6:** Multi-digit operations, decimals
- **Groep 7:** Percentages, ratios, negative numbers
- **Groep 8:** Advanced conversions, complex word problems

**Error types:**
- `rekenfout_basis` - Basic calculation error
- `conceptfout` - Conceptual misunderstanding
- `procedurele_fout` - Wrong method/procedure
- `afleesfout` - Misread the question
- `eenhedenfout` - Units confusion

## üîÑ Batch Processing Tips

### Similar Exercise Batching

Many templates contain similar exercises that can be enhanced in batches:

**Math examples:**
- "Tafel van 6" through "Tafel van 10" ‚Üí Similar feedback patterns
- "Optellen tot 1000" exercises ‚Üí Adapt number-specific feedback
- "Procent 20%" through "Procent 75%" ‚Üí Same structure, different percentages

**Reading examples:**
- Multiple informative articles ‚Üí Similar structure analysis patterns
- Story-based exercises ‚Üí Similar narrative comprehension feedback

**Strategy:**
1. Complete first exercise with full detail
2. Copy feedback structure
3. Adapt specific details (numbers, text references, etc.)
4. Review for accuracy

**Speed gain:** 2-3x faster for similar exercises
